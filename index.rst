PyGeode 0.6 documentation
===================================

Welcome! PyGeode is a python package intended to simplify the manipulation, analysis and
visualization of geophysical datasets, such as those generated by climate models. The documentation
is still a work in progress, but if you're new to PyGeode, the tutorial is a good place to start.
